# 🍽️ SRK Restaurant - Digital Menu

A lightweight, responsive digital menu for **SRK Restaurant**. This project focuses on a clean UI to help customers browse dishes and descriptions effortlessly.

## 📋 Features
* **Digital Menu:** A curated list of appetizers, main courses, and desserts.
* **Detailed Descriptions:** Each item includes ingredients and flavor profiles.
* **Responsive Layout:** Easy to read on both mobile phones and desktops.

## 🛠️ Tech Stack
* **HTML5** - Structure and Content
* **CSS3** - Styling and Layout
